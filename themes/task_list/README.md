### Task list

A mobile application displaying user's tasks.

#### Api Docs

https://jsonplaceholder.typicode.com/

#### Features

These are the pages / features the app should have:

* Splash;
* Login;
    * if backend sign in/up is not available, make a local auth with cache;
* Task list (home);
* Task details;
* Favorite tasks;
* Task not found;
* No network;
