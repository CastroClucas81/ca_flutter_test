### Anime List

A mobile application displaying information about various animes.

#### Api Docs

https://anilist.gitbook.io/anilist-apiv2-docs

#### Features

These are the pages / features the app should have:

* Splash;
* Login;
* Anime list (home);
* Anime details;
* Favorite animes;
* Anime not found;
* No network;
